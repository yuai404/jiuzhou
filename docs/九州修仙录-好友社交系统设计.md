# 九州修仙录 - 好友社交系统设计

> 版本：v0.1  
> 日期：2026-02-01  
> 目标：设计好友、私聊、师徒、仇人等社交系统。

## 1. 设计原则

- **社交核心**：好友是组队、交易、互动的基础
- **互动激励**：好友互动有奖励
- **关系多样**：好友、师徒、仇人等多种关系
- **隐私保护**：可设置隐私选项

## 2. 好友系统

### 2.1 好友功能

| 功能 | 说明 |
|------|------|
| 添加好友 | 发送/接受好友请求 |
| 删除好友 | 单方面删除 |
| 好友列表 | 查看在线状态、位置 |
| 好友分组 | 自定义分组 |
| 好友上限 | 基础50人，VIP可扩展 |

### 2.2 好友互动

| 互动 | 效果 | 次数限制 |
|------|------|----------|
| 赠送体力 | 送/收体力 | 每日各20次 |
| 切磋 | 友好PVP | 无限制 |
| 组队邀请 | 快速组队 | 无限制 |
| 传送 | 传送到好友位置 | 每日3次 |

### 2.3 亲密度

| 等级 | 亲密度 | 特权 |
|------|--------|------|
| 点头之交 | 0-99 | 基础功能 |
| 普通好友 | 100-499 | 传送次数+1 |
| 挚友 | 500-999 | 组队经验+5% |
| 知己 | 1000+ | 组队经验+10%、专属称号 |

## 3. 数据结构

### 3.1 好友关系表 (friend_relation)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigserial | 主键 |
| character_id | int | 角色ID |
| friend_id | int | 好友角色ID |
| group_name | varchar(32) | 分组名称 |
| intimacy | int | 亲密度 |
| remark | varchar(64) | 备注名 |
| created_at | timestamptz | 添加时间 |

### 3.2 好友请求表 (friend_request)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigserial | 主键 |
| from_id | int | 发送者 |
| to_id | int | 接收者 |
| message | varchar(128) | 验证消息 |
| status | varchar(16) | 状态（pending/accepted/rejected） |
| created_at | timestamptz | 发送时间 |
| handled_at | timestamptz | 处理时间 |

### 3.3 黑名单表 (blacklist)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigserial | 主键 |
| character_id | int | 角色ID |
| blocked_id | int | 被拉黑角色ID |
| reason | varchar(128) | 原因 |
| created_at | timestamptz | 拉黑时间 |

## 4. 师徒系统

### 4.1 拜师条件

- 徒弟：境界低于炼炁化神
- 师父：境界高于徒弟2个大境界
- 每人最多收3个徒弟

### 4.2 师徒奖励

| 事件 | 师父奖励 | 徒弟奖励 |
|------|----------|----------|
| 拜师成功 | 灵石×100 | 新手礼包 |
| 徒弟升境界 | 灵石×50 | 经验加成 |
| 徒弟出师 | 出师礼包 | 出师礼包 |

### 4.3 师徒任务

- 每日师徒任务
- 组队经验加成
- 师父指导（临时属性加成）

## 5. 仇人系统

### 5.1 仇人来源

- PVP击杀
- 抢夺资源
- 手动添加

### 5.2 仇人功能

- 仇人列表
- 仇人位置追踪
- 复仇任务

## 6. 私聊系统

### 6.1 私聊功能

- 实时私聊
- 离线消息
- 消息记录（保存7天）
- 屏蔽功能

### 6.2 私聊表 (private_message)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigserial | 主键 |
| from_id | int | 发送者 |
| to_id | int | 接收者 |
| content | text | 内容 |
| is_read | boolean | 是否已读 |
| created_at | timestamptz | 发送时间 |

## 7. 附近玩家

### 7.1 功能

- 显示同地图玩家
- 快速添加好友
- 快速组队
- 快速交易

### 7.2 隐私设置

- 显示/隐藏位置
- 允许/禁止陌生人私聊
- 允许/禁止组队邀请

## 8. API设计

```
GET /api/friend/list              # 好友列表
POST /api/friend/request          # 发送好友请求
POST /api/friend/accept           # 接受请求
POST /api/friend/reject           # 拒绝请求
DELETE /api/friend/:id            # 删除好友
POST /api/friend/gift             # 赠送体力

GET /api/message/history          # 私聊记录
POST /api/message/send            # 发送私聊

POST /api/blacklist/add           # 拉黑
DELETE /api/blacklist/:id         # 取消拉黑

GET /api/nearby/players           # 附近玩家
```
