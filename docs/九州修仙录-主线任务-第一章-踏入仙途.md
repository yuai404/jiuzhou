# 第一章：踏入仙途

> 版本：v0.2

> 章节概述：玩家从一个普通凡人开始，在青云村遇到村长指引，踏上修仙之路。

---

## 章节背景

青云村，九州大陆东洲边陲的一个宁静小村，灵气稀薄但民风淳朴。许多修士从这里踏上修仙之路。

村外是青云村外的野外区域，有一些低级妖兽出没，是新手修士历练的好去处。更深处的密林中，传说有狼王盘踞，甚至有千年古树妖作祟……

玩家扮演的角色是青云村的一名年轻人，因一次意外的机缘，命运从此改变……

### 任务描述

> 你是青云村的一名普通年轻人。今日清晨，你在村口醒来，却发现自己做了一个奇怪的梦——梦中有一位白发老者对你说："命运的齿轮已经转动，去找村中广场的村长吧。"

### 任务目标

| 目标ID | 类型 | 描述 | 目标 | 数量 |
|--------|------|------|------|------|
| obj-1 | talk | 与引路童子对话 | npc-guide | 1 |
| obj-2 | reach | 前往村中广场 | room-village-center | 1 |
| obj-3 | talk | 与村长对话 | npc-village-elder | 1 |

### NPC对话

#### 接取对话（自动触发 - 村口）

```
【旁白】
你从梦中惊醒，发现自己躺在青云村的村口。
阳光透过树叶洒落，远处传来鸡鸣声。
那个奇怪的梦……白发老者的话还在耳边回响。

【系统提示】
与村口的引路童子对话，了解村中情况。
```

#### 与引路童子对话

```
【引路童子】
欢迎来到青云村！修行之路漫漫，先从认识这里开始吧。
（童子热情地指向北方）
村中广场在北边，村长大人就在那里。
他德高望重，见证了无数修士踏上修仙之路。
你若有疑问，尽管问我。

【选项】
1. "村长是什么人？" → 继续对话
2. "我该怎么开始修行？" → 继续对话

【引路童子】（选项1）
村长大人虽然只是养气期的修为，但他见多识广。
听说年轻时也是闯荡过江湖的人物呢！
你去找他，说不定会有意外收获。

【引路童子】（选项2）
修行之路，首先要找到引路人。
村长大人会指点你的，快去吧！
```

#### 与村长对话

```
【村长】
你来了。青云村虽小，却也暗流涌动。
（村长上下打量着你）
我看你今天气色不太对，是不是做了什么奇怪的梦？

【选项】
1. "您怎么知道？" → 继续对话
2. "没什么，只是路过。" → 继续对话

【村长】（选项1）
哈哈，老夫活了这么多年，什么没见过。
你身上有一股……特殊的气息。
（村长压低声音）
若想在这世道立足，先练胆，再练心。

【村长】
村外的南林小道有些野兔出没，虽然弱小，但对你来说正好练手。
（村长递给你一本泛黄的书册）
这是《基础拳法》，是最基础的武技，你先拿去修炼。
等你击败几只野兔，再来找我。
```
