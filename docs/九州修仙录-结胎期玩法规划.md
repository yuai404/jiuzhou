# 九州修仙录 - 炼炁化神·结胎期玩法规划

> 版本：v0.1  
> 日期：2026-02-15  
> 目标：补齐“采药后段 -> 结胎突破 -> 结胎稳态”完整内容闭环，作为后续“养神期”开放前的核心成长阶段。

---

## 1. 规划目标

1. 建立结胎期完整可玩循环：刷图 -> 副本 -> 材料 -> 功法/装备 -> 突破 -> 继续刷取。
2. 保持数值节奏“稳步抬升”，避免从采药期到结胎期出现断层式难度跳变。
3. 复用现有服务端能力与数据结构，优先通过 seed 配置落地，不引入新接口类型。

---

## 2. 范围边界

### 2.1 本期包含

1. 采药期后段过渡内容（地图、任务、两张过渡副本）。
2. 采药期 -> 结胎期突破条件与消耗重设（解除该段版本锁）。
3. 结胎期稳态内容（新地图、新副本、怪物、掉落、套装、日常/周常）。
4. 第四章主线（对应 `mq-chapter-4`）。

### 2.2 本期不包含

1. 新玩法系统代码（例如新战斗机制引擎、全新任务目标类型）。
2. 养神期及以上阶段的正式开放。
3. 大规模前端交互改版（仅使用现有 UI 流程承载新内容）。

---

## 3. 现状基线

1. 当前已实装内容主线集中在“炼己期”，并已铺设采药/结胎境界文本与部分功法层级门槛。
2. `realm_breakthrough.json` 中 `炼炁化神·采药期 -> 炼炁化神·结胎期` 仍带 `version_locked` 条目。
3. 副本种子当前已到 `dungeon_qi_cultivation_8.json`。
4. 主线章节目前已有 `main_quest_chapter1~3.json`，章节体系可直接扩展为 chapter 4。

---

## 4. 内容结构总览

结胎期内容按三段组织：

1. 采药后段过渡：引导玩家进入“胎息稳定”主题区，积累突破材料。
2. 结胎突破仪式：通过主线章节 + 副本通关 + 功法层级 + 材料达成突破。
3. 结胎稳态循环：开放高阶副本、装备套装与周常，形成长期养成池。

---

## 5. 主线章节规划（Chapter 4）

## 5.1 章节定义

1. 章节 ID：`mq-chapter-4`
2. 章节名：第四章（建议名：`胎息初成`）
3. 最低境界：`炼炁化神·采药期`
4. 解锁功能：新地图 + 过渡副本 + 结胎期主副本

## 5.2 段落规划（9 段）

1. `main-4-001`：到达新前哨并交谈（`reach + talk_npc`）。
2. `main-4-002`：清理采药后段怪群（`kill_monster`）。
3. `main-4-003`：采集与收集突破材料（`gather_resource + collect`）。
4. `main-4-004`：通关过渡副本 1（普通）。
5. `main-4-005`：通关过渡副本 2（普通）。
6. `main-4-006`：挑战过渡副本高难（困难）。
7. `main-4-007`：达成功法成长目标（`upgrade_technique`）。
8. `main-4-008`：突破到 `炼炁化神·结胎期`（`upgrade_realm`）。
9. `main-4-009`：回报并完章（`talk_npc`，章节收束）。

说明：第 8 段使用现有主线静态目标能力，无需新增目标类型。

## 9. 结胎突破规划

目标链路：`炼炁化神·采药期 -> 炼炁化神·结胎期`

## 9.1 条件建议

1. 修为经验：`exp >= 1,800,000`
2. 功法门槛：至少 4 门功法达到 7 层
3. 副本门槛：过渡副本困难难度累计通关（建议各 2 次）
4. 主线门槛：完成 `mq-chapter-4`
5. 材料门槛：`mat-taiyuan-jingpo`、`mat-jietai-dan`

## 9.2 消耗建议

1. 灵石：6800
2. 经验：1,800,000
3. 材料：`mat-taiyuan-jingpo` + `mat-jietai-dan`

## 9.3 奖励建议

1. 属性点：20
2. 核心属性百分比：血/灵/攻/防稳步提升
3. 附加属性：控制抗性小幅提升

说明：仅解除本段版本锁；`结胎期 -> 养神期` 保持锁定。

## 10. 任务池规划

## 10.1 采药后段

1. 日常 5 条：清剿、采集、过渡副本通关、材料提交。
2. 周常 4 条：副本累计、精英击杀、双地图巡历、突破材料周目标。

## 10.2 结胎稳态

1. 日常 3 条：结胎地图清剿、主副本日修、核心材料收集。
2. 周常 2 条：Boss 轮转挑战、套装养成材料周目标。

任务目标类型仅使用现有 `taskService` 已支持类型。

---

## 11. 数值节奏建议

1. 采药后段对炼己后段：约 +15%~25% 战斗强度。
2. 结胎稳态对采药后段：再 +12%~20%。
3. 单位体力产出：随难度提高而提升，噩梦档约为普通 1.8~2.0 倍。
4. 目标周节奏：普通玩家 1~2 周可达结胎突破门槛，活跃玩家 4~6 天可达。

---

## 12. 实施顺序

1. 文档与命名冻结（本文件）。
2. 新增 chapter4 主线与对话 seed。
3. 新增地图/NPC/怪物/掉落 seed。
4. 新增 3 个副本 seed（9~11）。
5. 新增材料、套装、任务 seed。
6. 调整 `realm_breakthrough.json`（解除结胎段版本锁）。
7. 统一校验并回归（`tsc -b` + 数据引用检查）。

---

## 13. 验收标准

1. 采药期角色可完整体验 chapter4 并触发突破目标。
2. 满足条件后可稳定突破至 `炼炁化神·结胎期`，扣费与奖励正确。
3. 新地图与副本可被静态配置加载并正确显示。
4. 新任务可推进并可领奖，不出现“目标类型不识别”。
5. 新掉落与套装引用无缺失、无悬挂 ID。

---

## 14. 风险与回滚

1. 风险：新 seed 间引用关系复杂，容易出现 ID 悬挂。
2. 风险：突破门槛过高导致卡关，或过低导致阶段寿命不足。
3. 回滚策略：按文件粒度回滚（优先回滚 `realm_breakthrough.json` 与 chapter4 入口配置），保证主流程可继续。
