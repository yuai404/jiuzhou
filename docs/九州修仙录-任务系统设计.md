# 九州修仙录 - 任务系统设计

> 版本：v0.1  
> 日期：2026-02-01  
> 目标：设计完整的任务系统，支持主线、支线、日常、悬赏、宗门任务与任务链。

## 1. 设计原则

- **引导新手**：主线任务引导玩家熟悉核心玩法
- **持续目标**：日常/周常提供稳定的游戏目标
- **叙事载体**：支线任务承载世界观与故事
- **奖励驱动**：任务是重要的资源产出渠道
- **可扩展**：支持活动任务、限时任务热更新

## 2. 任务分类

### 2.1 按类型分类

| 类型 | 说明 | 重置周期 | 特点 |
|------|------|----------|------|
| 主线任务 | 核心剧情线 | 不重置 | 引导、高奖励、解锁功能 |
| 支线任务 | 世界观扩展 | 不重置 | 故事、探索、隐藏内容 |
| 日常任务 | 每日目标 | 每日5点 | 稳定产出、快速完成 |
| 周常任务 | 每周目标 | 每周一5点 | 高价值奖励 |
| 悬赏任务 | 随机生成 | 完成后刷新 | 可选难度、灵活 |
| 宗门任务 | 宗门专属 | 每日/每周 | 宗门贡献、集体目标 |
| 活动任务 | 限时活动 | 活动期间 | 特殊奖励 |

### 2.2 按目标类型

| 目标类型 | 说明 | 示例 |
|----------|------|------|
| kill | 击杀怪物 | 击杀10只妖狼 |

| talk | 对话NPC | 与长老对话 |

## 4. 任务流程

### 4.1 接取流程

```
1. 检查前置条件（境界、前置任务、等级）
2. 检查任务槽位（日常/悬赏有数量限制）
3. 创建任务实例（character_quest）
4. 初始化目标进度
5. 触发接取对话/动画
6. 发送任务接取通知
```

### 4.2 进度更新流程

```
1. 游戏事件触发（击杀、采集、对话等）
2. 匹配进行中的任务目标
3. 更新目标进度
4. 检查是否完成所有目标
5. 完成则更新任务状态
6. 发送进度/完成通知
```

### 4.3 领取奖励流程

```
1. 验证任务状态为completed
2. 检查背包空间
3. 发放奖励（经验/货币/物品）
4. 更新任务状态为claimed
5. 触发后续任务（如有）
6. 触发成就检测
```

### 11.4 事件集成

```typescript
// server/src/events/questEventHandler.ts
import { gameEvents } from './GameEventEmitter.js';
import { GameEventType } from './GameEventTypes.js';
import { updateQuestProgress } from '../services/questService.js';

export function registerQuestEventHandlers() {
  // 击杀怪物
  gameEvents.on(GameEventType.MONSTER_KILLED, async (event) => {
    await updateQuestProgress(event.characterId, 'kill', event.data.monsterId);
    await updateQuestProgress(event.characterId, 'kill', '*'); // 通配符
  });
  
  // 获得物品
  gameEvents.on(GameEventType.ITEM_OBTAINED, async (event) => {
    await updateQuestProgress(event.characterId, 'collect', event.data.itemDefId, event.data.quantity);
  });
  
  // 对话NPC
  gameEvents.on(GameEventType.NPC_TALKED, async (event) => {
    await updateQuestProgress(event.characterId, 'talk', event.data.npcId);
  });
  
  // 通关副本
  gameEvents.on(GameEventType.DUNGEON_CLEARED, async (event) => {
    await updateQuestProgress(event.characterId, 'dungeon', event.data.dungeonId);
  });
  
  // 功法升级
  gameEvents.on(GameEventType.TECHNIQUE_UPGRADED, async (event) => {
    await updateQuestProgress(event.characterId, 'upgrade', event.data.techniqueId);
  });
}
```
