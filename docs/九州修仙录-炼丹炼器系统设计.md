# 九州修仙录 - 炼丹炼器系统设计

> 版本：v0.1  
> 日期：2026-02-01  
> 目标：设计完整的生活技能系统，包含炼丹、炼器、采集与配方系统。

## 1. 设计原则

- **产出闭环**：采集→炼制→使用/交易形成完整循环
- **技能深度**：熟练度、配方解锁、品质控制提供成长线
- **经济价值**：高品质产出有交易价值
- **时间投入**：炼制需要时间，可加速
- **风险收益**：高难度配方有失败风险，但产出更好

### 3.2 材料消耗结构

```typescript
interface RecipeMaterial {
  item_def_id: string;    // 材料ID
  quantity: number;       // 数量
  quality_min?: string;   // 最低品质要求
  consume_on_fail?: boolean;  // 失败是否消耗（默认true）
}
```

### 3.3 产出配置结构

```typescript
interface RecipeProduct {
  item_def_id: string;    // 产出物品ID
  quantity_base: number;  // 基础数量
  quantity_bonus?: number; // 额外数量（暴击时）
  
  // 品质规则
  quality_mode: 'fixed' | 'random' | 'skill_based';
  quality_fixed?: string;  // 固定品质
  quality_weights?: Record<string, number>;  // 随机权重
  
  // 装备特殊规则
  equip_options?: {
    affix_pool_id?: string;  // 词条池
    quality_weights?: Record<string, number>;
  };
}
```

## 4. 炼丹系统

### 4.1 丹药分类

| 分类 | 效果 | 示例 |
|------|------|------|
| 回复丹 | 恢复气血/灵气 | 回气丹、续命丹 |
| 增益丹 | 临时属性提升 | 力量丹、敏捷丹 |
| 突破丹 | 辅助境界突破 | 筑基丹、金丹 |
| 解毒丹 | 解除负面状态 | 解毒丹、清心丹 |
| 特殊丹 | 特殊效果 | 易容丹、隐身丹 |

### 4.2 炼丹品质

| 品质 | 效果倍率 | 出现概率（基础） |
|------|----------|------------------|
| 下品 | 80% | 40% |
| 中品 | 100% | 35% |
| 上品 | 120% | 20% |
| 极品 | 150% | 5% |

### 4.3 炼丹成功率

```
基础成功率 = 配方基础率 - (配方难度 - 技能等级) × 5%
最终成功率 = 基础成功率 + 熟练度加成 + 装备加成
```

### 4.4 品质计算

```
品质权重 = 基础权重 × (1 + 技能等级加成 + 熟练度加成)
最终品质 = 按权重随机
```

## 5. 炼器系统

### 5.1 装备制作

| 类型 | 产出 | 特点 |
|------|------|------|
| 武器 | 各类武器 | 高攻击词条 |
| 防具 | 头/衣/手/裤 | 高防御词条 |
| 饰品 | 项链/饰品 | 特殊词条 |
| 法宝 | 法宝 | 技能效果 |

### 5.2 词条生成

炼器产出的装备词条规则：
- 品质决定词条数量（与掉落装备一致）
- 技能等级影响词条Tier上限
- 熟练度影响高Tier词条概率

### 5.3 特殊材料

| 材料类型 | 效果 |
|----------|------|
| 主材料 | 决定装备基础属性 |
| 辅材料 | 影响词条类型倾向 |
| 特殊材料 | 保证特定词条/品质 |

## 6. 采集系统

### 6.1 采集流程

```
1. 到达采集点
2. 检查技能等级/境界
3. 开始采集（消耗时间）
4. 计算采集结果（掉落池）
5. 获得材料
6. 采集点进入冷却
```

### 6.2 采集加成

| 来源 | 效果 |
|------|------|
| 技能等级 | 采集速度+、稀有率+ |
| 采集工具 | 采集速度+、数量+ |
| 宗门建筑 | 全局加成 |
| VIP | 采集速度+ |

### 6.3 稀有采集

- 隐藏采集点：需要探索发现
- 限时采集点：特定时间出现
- 竞争采集点：多人抢夺

## 7. 配方系统

### 7.1 配方获取

| 方式 | 说明 |
|------|------|
| 默认解锁 | 基础配方自动获得 |
| 技能升级 | 达到等级自动解锁 |
| 配方书 | 使用配方书学习 |
| 任务奖励 | 完成任务获得 |
| NPC购买 | 商店购买 |
| 悟道 | 随机领悟（高级） |

### 7.2 配方升级

部分配方可升级：
- 提高成功率
- 提高品质概率
- 减少材料消耗
- 减少制作时间

### 8.1 队列规则

- 可排队多个任务
- 可暂停/取消
- 离线继续制作

### 8.2 加速机制

| 方式 | 效果 |
|------|------|
| 灵石加速 | 立即完成 |
| 加速道具 | 减少剩余时间 |
| VIP | 制作速度+20% |

## 9. API设计

### 9.1 配方列表

```
GET /api/craft/recipes
Query: { skill_type, category, learned_only }
Response: { recipes: RecipeInfo[] }
```

### 9.2 开始制作

```
POST /api/craft/start
Body: { recipe_id, quantity }
Response: { success, queue_id, end_time }
```

### 9.3 领取成品

```
POST /api/craft/claim
Body: { queue_id }
Response: { success, products: ProductResult[] }
```

### 9.4 采集

```
POST /api/gather/start
Body: { node_id }
Response: { success, end_time }

POST /api/gather/claim
Body: { gather_id }
Response: { success, items }
```
