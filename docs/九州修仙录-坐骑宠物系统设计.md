# 九州修仙录 - 坐骑宠物系统设计

> 版本：v0.1  
> 日期：2026-02-01  
> 目标：设计坐骑与宠物系统，提供移动加成、战斗辅助与收集养成玩法。

## 1. 设计原则

- **功能明确**：坐骑主移动，宠物主战斗
- **养成深度**：等级、技能、进化提供成长线
- **收集乐趣**：多种类、稀有度、图鉴系统
- **外观展示**：坐骑/宠物是身份象征

## 2. 坐骑系统

### 2.1 坐骑功能

| 功能 | 说明 |
|------|------|
| 移动加速 | 地图移动速度提升 |
| 属性加成 | 提供永久属性加成 |
| 外观展示 | 骑乘外观 |
| 技能效果 | 特殊坐骑有技能 |

### 2.2 坐骑品质

| 品质 | 基础速度 | 属性加成 | 技能槽 |
|------|----------|----------|--------|
| 黄 | +20% | 1条 | 0 |
| 玄 | +40% | 2条 | 1 |
| 地 | +60% | 3条 | 2 |
| 天 | +100% | 4条 | 3 |

### 2.3 坐骑获取

- 任务奖励
- 副本掉落
- 商城购买
- 活动兑换
- 野外捕捉

## 3. 宠物系统

### 3.1 宠物功能

| 功能 | 说明 |
|------|------|
| 战斗辅助 | 参与战斗，自动释放技能 |
| 属性加成 | 提供属性加成 |
| 采集辅助 | 自动采集（可选） |
| 外观展示 | 跟随外观 |

### 3.2 宠物类型

| 类型 | 定位 | 特点 |
|------|------|------|
| 攻击型 | 输出 | 高伤害技能 |
| 防御型 | 坦克 | 嘲讽、护盾 |
| 辅助型 | 治疗 | 回复、增益 |
| 平衡型 | 均衡 | 各项平均 |

### 3.3 宠物技能

- 主动技能：战斗中自动释放
- 被动技能：永久生效
- 天赋技能：品质决定

## 4. 数据结构

### 4.1 坐骑/宠物定义表 (mount_pet_def)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | varchar(64) | ID |
| name | varchar(64) | 名称 |
| type | varchar(16) | 类型（mount/pet） |
| sub_type | varchar(32) | 子类型 |
| quality | char(1) | 品质 |
| description | text | 描述 |
| model | varchar(256) | 模型 |
| base_attrs | jsonb | 基础属性 |
| skills | jsonb | 技能列表 |
| speed_bonus | int | 速度加成（坐骑） |
| obtain_type | varchar(32) | 获取方式 |
| enabled | boolean | 是否启用 |

### 4.2 角色坐骑/宠物表 (character_mount_pet)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigserial | 主键 |
| character_id | int | 角色ID |
| def_id | varchar(64) | 定义ID |
| type | varchar(16) | 类型 |
| name | varchar(64) | 自定义名称 |
| level | int | 等级 |
| exp | int | 经验 |
| skills | jsonb | 已学技能 |
| is_active | boolean | 是否出战 |
| obtained_at | timestamptz | 获得时间 |

## 5. 养成系统

### 5.1 等级提升

- 经验来源：战斗、喂养、任务
- 等级上限：与角色境界挂钩
- 属性成长：每级提升基础属性

### 5.2 技能学习

- 技能书学习
- 等级解锁
- 进化获得

### 5.3 进化系统

- 消耗进化材料
- 提升品质上限
- 解锁新技能槽

## 6. API设计

```
GET /api/mount/list          # 坐骑列表
POST /api/mount/ride         # 骑乘
POST /api/mount/upgrade      # 升级

GET /api/pet/list            # 宠物列表
POST /api/pet/summon         # 出战
POST /api/pet/feed           # 喂养
POST /api/pet/learn-skill    # 学习技能
```
