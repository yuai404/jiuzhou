# 九州修仙录 - 休闲玩法系统设计

> 版本：v0.1  
> 日期：2026-02-01  
> 目标：设计轻松有趣的休闲玩法，丰富游戏体验，提供社交互动场景。

## 1. 设计原则

- **轻松愉快**：无压力的休闲体验
- **社交互动**：提供玩家互动场景
- **额外收益**：休闲玩法也有奖励
- **碎片时间**：适合短时间游玩

## 3. 灵田系统详细设计

### 3.1 灵田数据表 (spirit_farm)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigserial | 主键 |
| character_id | int | 角色ID |
| plot_count | int | 已开垦格子数 |
| plots | jsonb | 格子状态 |
| last_water_time | timestamptz | 上次浇水时间 |
| daily_water_count | int | 今日浇水次数 |

### 3.2 格子状态结构

```typescript
interface FarmPlot {
  index: number;           // 格子索引
  status: 'empty' | 'planted' | 'growing' | 'harvestable';
  seed_id?: string;        // 种子ID
  plant_time?: number;     // 种植时间戳
  water_count?: number;    // 浇水次数
  fertilize_count?: number; // 施肥次数
  harvest_time?: number;   // 可收获时间
}
```

### 3.3 种子定义表 (seed_def)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | varchar(64) | 种子ID |
| name | varchar(64) | 名称 |
| grow_time_sec | int | 生长时间（秒） |
| harvest_item_id | varchar(64) | 收获物品ID |
| harvest_qty_min | int | 最小收获数量 |
| harvest_qty_max | int | 最大收获数量 |
| water_bonus | int | 浇水加速（秒/次） |
| fertilize_bonus | int | 施肥加速（秒/次） |

### 3.4 灵田API

```

### 4.4 钓鱼流程

```
1. 选择钓鱼点
2. 选择鱼竿和鱼饵
3. 抛竿等待（3-10秒随机）
4. 鱼咬钩提示
5. 点击收杆（时机影响成功率）
6. 计算结果（鱼种、重量）
7. 获得奖励
```

### 4.5 钓鱼API

```

GET /api/fishing/records       # 钓鱼记录
